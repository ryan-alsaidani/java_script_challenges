Rock Paper Scissors Game

This repository contains my implementation of the Rock Paper Scissors game using HTML, CSS, and JavaScript. It is designed for learning, experimentation, and improving my coding skills.

Repository

The repository is hosted here: "Rock Paper Scissors Game Repository" (https://github.com/ryan-alsaidani/rock_paper_scissors_game)

Description

- Classic single-player Rock Paper Scissors game (player vs. computer).
- Dynamic selection and UI updates.
- Keeps score of the player and the computer using "localStorage".
- Click on a previously used selection to switch players without modifying the current choice.
- Clear, modular code structure for each task.

Features

- Player chooses Rock, Paper, or Scissors against the computer.
- Score tracking across rounds.
- Dynamic updates after each turn.
- Easy to follow and modify for experimentation.

Installation

1. Clone the repository:
git clone https://github.com/ryan-alsaidani/rock_paper_scissors_game.git
2. Open "index.html" in your browser.
3. Start playing the game.

Usage

- Click on Rock, Paper, or Scissors to make your choice.
- The computer automatically makes its choice.
- Clicking a used selection switches the player without changing the previous choice.
- The game announces the winner of each round and updates the score.
- Use Reset buttons to restart the scores.
